# 学生选课通知

各位同学：

为做好体育选课工作，请认真阅读并按要求完成操作，具体通知如下。

一、登录方式
1. 登录网址：www.jsqyty.com。
2. 登录账号和初始密码请联系班主任领取。

二、登录与首次使用
1. 首次登录后，如系统提示修改密码，请先完成密码修改。
2. 首次登录后，请及时绑定本人常用邮箱，并完成邮箱验证。该邮箱将用于密码找回、账号安全验证和接收相关通知。

三、忘记密码处理
1. 在登录页面点击“忘记密码”。
2. 按提示输入账号和已绑定邮箱，提交找回申请。
3. 登录绑定邮箱，打开系统发送的重置链接。
4. 按要求设置新密码后，返回系统重新登录。
5. 如未绑定邮箱或无法完成找回，请及时联系老师或管理员处理。

四、选课操作流程
1. 登录成功后，进入“学生选课”或“课程列表”页面。
2. 查看课程名称、任课教师、适用年级及剩余名额等信息。
3. 结合个人兴趣和学校安排，在规定时间内完成选课。
4. 第一轮按要求填报志愿；第二轮在开放后选择尚有名额的课程。
5. 提交后，请及时进入“我的课程”或课程列表页面查看选课结果。

五、重要说明
1. 请务必在规定时间内完成选课，逾期可能影响课程安排。
2. 第二轮结束后，仍未选课的学生，系统将随机分配到仍有剩余名额的项目中。
3. 随机分配完成后，请及时登录系统查看最终结果。
4. 系统显示的最终课程结果为正式选课结果，请同学们按要求参加学习。

六、注意事项
1. 请妥善保管账号和密码，不得转借他人使用。
2. 请确保绑定邮箱真实有效并由本人使用。
3. 每次完成操作后，请重新查看页面确认是否提交成功。
4. 如遇系统异常、无法登录或无法选课，请及时联系老师或管理员。

请各位同学按要求认真完成相关操作。

特此通知。
