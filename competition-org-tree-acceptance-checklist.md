# 组织树与赛事系统验收清单

## 一、验收目标

本文档用于对当前 `feature/competition-system` 分支上的“组织树 + 赛事系统 + 学生账号兼容改造”进行业务验收和联调验收。

本次验收重点不是检查代码实现细节，而是确认以下问题：

1. 市级、区县、学校三级组织管理是否符合预期
2. 学生账号兼容方案是否与赛事系统协同正常
3. 赛事主流程是否可以完整走通
4. Web 管理端页面是否达到联调可用状态
5. 是否具备进入下一阶段持续打磨或试运行的条件
